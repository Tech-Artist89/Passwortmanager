Initialesieren der virtuellen Umgebung und Installation der requierement.txt für die benötigten Abhängigkeiten.
Es wird geraten, vor der installation der Abhängigkeiten eine virtuelle Umgebung zu erstellen, da sonst die Funktionalität nicht gewährleistet werden kann!

Erstellen einer virtuellne Umgebung unter Python:

1. Erstellen einer Virtuellen Umgebung:
   python -m venv venv

2. Virtuelle Umgebung aktivieren:
   .\venv\Scripts\Activate.ps1

3. Abhängigkeiten installieren:
   pip install -r requirements.txt

4. Programm ausführen:
   python main.py
